# student-registry-app – webalkalmazás

A **student-registry-app** egy webes alkalmazás, amely tanárok, diákok és órák kezelésére szolgál.  
A rendszer célja, hogy egyszerűen kezelhető felületet biztosítson az órák létrehozásához, szervezéséhez és a diákok jelentkezéséhez.

Az alkalmazás három fő szerepkört tartalmaz:

- **Adminisztrátor**
- **Tanár**
- **Diák**

## Adminisztrátor
- Tanárok létrehozása és kezelése
- Diákok létrehozása

## Tanár
- Új órák létrehozása
- Órák törlése
- Órák időpontjának módosítása vagy áthelyezése
- Az órák megjelenítése a naptárban
- Diákok létrehozása

## Diák
- Jelentkezés egy órára
- A felvett óra automatikusan megjelenik a saját naptárában
- Jelentkezés lemondása egy óráról

A projekt mintapéldájában a tanított tantárgy neve **Sötét varázslatok kivédése**.
